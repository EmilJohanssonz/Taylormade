# Taylormade Website

This project is a simple, static website for **Taylormade**, a golf equipment company. The website showcases products, athletes, and includes navigation, news, and login functionalities.

## Features

1. **Header**: Displays the Taylormade logo and an introduction text about the brand.
2. **Navigation Menu**: A collapsible side menu with links to various golf equipment categories.
3. **Main Content**: 
   - **News Section**: Shows the latest Taylormade golf clubs with images and descriptions.
   - **Product Highlight Section**: Information about building the perfect golf club with images of different types of clubs.
   - **Team Taylormade**: Showcases famous professional golfers who use Taylormade equipment, including Tiger Woods and Rory McIlroy.
4. **Login Form**: Simple login form with fields for username and password.
5. **Social Media Links**: Links to Taylormadeâ€™s social media accounts on Facebook, Instagram, and YouTube.
